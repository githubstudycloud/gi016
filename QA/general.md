# 通用技术 QA

## [Q] 如何在 Markdown 中记录代码块？

- **日期**: 2024-01-20
- **标签**: #markdown #guide

### [A] 回答

使用三个反引号 \`\`\` 包裹代码。

例如：

\`\`\`javascript
console.log('Hello');
\`\`\`

## [Q] 为什么推荐用 Git 管理笔记？

- **日期**: 2024-01-20
- **标签**: #git #knowledge-base

### [A] 回答

1. **历史回溯**：可以查看以前的修改记录。
2. **云端同步**：推送到 GitHub/GitLab，防止丢失。
3. **多端协作**：方便在不同机器上同步。
