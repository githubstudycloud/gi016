# 个人 QA 知识库

这是一个基于 Markdown 的轻量级个人知识库模板。

## 目录结构

- `QA/` : 存放具体的问题与解答
  - `template.md` : 记录新问题的模板
  - `general.md` : 示例文件

## 如何使用

1. **新建记录**：复制 `QA/template.md` 或直接在对应分类文件中添加。
2. **搜索**：使用 IDE 的全局搜索 (Ctrl+Shift+F) 快速查找问题。
3. **版本管理**：建议使用 Git 进行版本控制，确保历史记录不丢失。

## 推荐工具

- **编辑器**: Trae / VS Code
- **预览**: Markdown Preview Enhanced (插件)
- **版本控制**: Git
