# AI 规则配置指南索引

本目录包含了将 **6A 工作流 (项目规则)** 和 **5S 敏捷规范 (个人规则)** 应用于主流 AI 编程工具的详细指南。

## 指南列表

- **[Trae 设置指南](./Trae_Setup.md)**: 适用于 Trae 编辑器，利用 Project Rules 和 Personal Rules 功能。
- **[Cursor 设置指南](./Cursor_Setup.md)**: 适用于 Cursor 编辑器，利用 `.cursorrules` 和全局设置。
- **[Claude Code 设置指南](./Claude_Code_Setup.md)**: 适用于 Anthropic Claude Code CLI，利用 `CLAUDE.md`。
- **[Gemini CLI 设置指南](./Gemini_CLI_Setup.md)**: 适用于 Google Gemini CLI，利用 `.gemini/system.md`。
- **[Codex CLI 设置指南](./Codex_CLI_Setup.md)**: 适用于 OpenAI Codex 相关 CLI 工具，利用配置文件或参数。

## 快速开始

1. 确定你使用的工具。
2. 点击上方对应的链接查看配置步骤。
3. 将上层目录中的 `project_rules_6A.md` 和 `personal_rules_5S.md` 内容应用到你的工具中。
